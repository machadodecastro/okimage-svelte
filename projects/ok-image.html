Image preprocessing with OK Image

A free image preprocessing tool to apply Data Augmentation techniques

Enjoy

Make your image preprocessing tasks easier.

Bitwise

Bitwise operations are used in image manipulation and important for extracting essential parts in the image.

Bitwise operations included are : Bitwise OR, AND, NOT and XOR.

Blending

The Blending of images consists of adding two images by giving a specific weight value to each image so that it gives a feeling of one image overlapping the other.

Blurring

Image blurring is achieved by convolving the image with a low-pass filter kernel. It is useful for removing noise.

Blur functions included are: Blur, Box Filter, Gaussian Blur and Median Blur.

Masking

Image masking allows us to focus only on the portions of the image that interests us.

Options included are: Circle and Rectangle Mask.

Histograms

Histograms capture the frequency distribution of a set of data. And it turns out that examining these frequency distributions is a very nice way to build simple image processing techniques.

Functions included are: Color and Grayscale histograms

Contrast Stretching

Contrast stretching is a simple image enhancement technique that attempts to improve the contrast in an image by 'stretching' the range of intensity values it contains to span a desired range of values, the full range of pixel values that the image type concerned allows.

Cropping

Cropping an image is one of the most basic image operations that is used in image preprocessing.

Cropping can be used to make an image smaller (in pixels) and/or to change the aspect ration (length to width) of the image.

Dilation

This operation consists of convolving an image with some kernel, which can have any shape or size.

Difference of Gaussians (DoG)

Difference of Gaussians (DoG) is calculated as the difference between two smoothed versions of an image obtained by applying two Gaussian kernels of different standard deviations (sigma) on that image.

Edge Cascade

Edge detection is an image-processing technique, which is used to identify the boundaries (edges) of objects, or regions within an image. Edges are among the most important features associated with images.

Erosion

Dilation adds pixels to the boundaries of objects in an image, while erosion removes pixels on object boundaries. The number of pixels added or removed from the objects in an image depends on the size and shape of the structuring element used to process the image.

Flipping

Flipping images helps increasing the dataset with the same image in different axes.

Gamma Correction

Gamma correction can be used to correct the brightness of an image by using a non linear transformation between the input values and the mapped output values.

Grayscale

Grayscaling is the process of converting an image from other color spaces e.g. RGB, CMYK, HSV, etc. to shades of gray. It varies between complete black and complete white.

Importance of grayscaling: Dimension reduction, Model complexity reduction and other algorithms compatibility because many algorithms are customized to work only on grayscale images .

Rotation

Image rotation is a common image processing routine with applications in matching, alignment, and other image-based algorithms.

The method is used to rotate a 2D array in multiples of 90 degrees.
